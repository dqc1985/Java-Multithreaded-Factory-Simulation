# Java Multithreaded Factory Simulation

## Overview

This project is a Java-based multithreaded factory production simulation that demonstrates object-oriented programming concepts, concurrency, synchronization, interfaces, and thread management.

The application simulates multiple factory production lines running simultaneously using Java threads. Each product type has its own production process and timing logic.

---

## Features

- Multi-threaded production simulation
- Java concurrency and synchronization
- ReentrantLock usage
- Interface-based design
- Separate product classes
- Production summary reporting
- Object-oriented programming principles
